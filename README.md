# PlovdivMC Website

[![License: MIT](https://img.shields.io/badge/license-MIT-green.svg)](https://opensource.org/licenses/MIT)

Professional static website for the **Plovdiv Minecraft Server**.  
Provides server info, online status, gameplay sections, and Discord integration. Deployable on **Netlify** or similar static hosting.
